## 🔍 Troubleshooting common issues

If the game still shows an error message, try these steps in order.

### Check game files in Steam
Steam often detects corrupt files automatically. Right-click Baldur’s Gate 3 in your library. Select Properties, go to Installed Files, and click Verify integrity of game files. Wait for this process to complete before you run the fix tool again.

### Update your graphics drivers
Outdated drivers cause many DirectX errors. Visit the website for your graphics card manufacturer—either NVIDIA, AMD, or Intel—and download the latest drivers for your specific hardware model. Restart your computer after the driver update finishes.

### Run as administrator
Right-click the game shortcut on your desktop. Choose Run as administrator. This provides the game with the permissions it needs to access DirectX components.
